# 南宁白话常用字读音表

上述资料包括一份南宁白话常用字读音表，还在不断修订，其他都是广州话字词典，用于对比。

不需要注册，右上方有“下载”按钮，点击即可下载。

使用字词典之前需要安装外挂类词典软件：

一、PC上可以使用GoldenDict，下载和使用方法见[这里](http://www.360doc.com/content/17/0130/05/89866_625374607.shtml)。「生僻字的显示要求电脑安装有相应的字体，例如花园明朝体与宋体-方正超大字符集」

二、移动端推荐使用欧路词典，可以在应用市场搜索到并下载下来，安装好欧路词典后，只要把上面的mdx与mdd文件放到手机的eudb_en文件夹即可。「需要显示生僻字可以安装深蓝词典，使用[外挂字体](http://www.ssdlsoft.com/bluedict/help/jiqiao/)」

![安卓手机的文件管理夹](http://oyh64ik7j.bkt.clouddn.com/%E6%96%87%E4%BB%B6.png)

三、关于这份读音表的说明参见[这里](https://www.pdawiki.com/forum/forum.php?mod=viewthread&tid=23065&extra=page%3D1)。

关于南宁白话的基础知识参看[南宁白话小册子](https://leimaau.gitbooks.io/nnbh/content/)
